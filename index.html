POLÍTICA DE PRIVACIDADE — LASO FS (Folha de Serviços)

1) Quem somos

FASE PILOTO: Este é um pré-lançamento da aplicação sob controlo do desenvolvedor.
Não é um canal oficial da LASO. Durante esta fase devem ser utilizados apenas dados de teste.

Responsável pelo tratamento (fase piloto):
Desenvolvedor do projeto LASO FS.

Contacto de suporte:
app.fs.piloto@gmail.com

A aplicação LASO FS (Folha de Serviços) destina-se ao registo e gestão de folhas de serviço e outras informações operacionais relacionadas com atividades profissionais.

2) Que dados podem ser tratados

A aplicação pode tratar dados introduzidos manualmente pelo utilizador no contexto de trabalho, por exemplo:

- registos de folhas de serviço
- ocorrências operacionais
- informação associada à atividade profissional
- outros dados necessários ao processo operacional

Nesta fase piloto, os dados são inseridos manualmente pelo utilizador para fins de teste e validação funcional da aplicação.

A aplicação não inclui publicidade e não realiza venda de dados pessoais.

3) Finalidades do tratamento

Os dados inseridos na aplicação podem ser utilizados para:

- registo e gestão de folhas de serviço
- suporte técnico e resolução de problemas
- melhoria funcional da aplicação
- validação operacional durante a fase piloto

4) Partilha de dados

Nesta fase piloto, a aplicação não envia automaticamente dados para servidores externos.

Qualquer partilha de dados ocorre apenas quando o próprio utilizador decide exportar ou comunicar a informação fora da aplicação, por exemplo através de email ou outros meios definidos pela organização que utiliza a aplicação.

Os dados não são partilhados com terceiros para fins de marketing.

5) Retenção de dados

Os dados são mantidos apenas pelo tempo necessário para cumprir os objetivos operacionais da aplicação ou eventuais obrigações legais aplicáveis.

Durante a fase piloto, os dados são utilizados exclusivamente para testes e validação do funcionamento da aplicação.

6) Segurança

São aplicadas medidas técnicas e organizacionais razoáveis para proteger os dados contra:

- acesso não autorizado
- alteração indevida
- perda ou destruição de informação

Apesar destas medidas, nenhum sistema informático pode garantir segurança absoluta.

7) Direitos dos utilizadores (RGPD)

Nos termos do Regulamento Geral de Proteção de Dados (RGPD), os utilizadores podem solicitar:

- acesso aos seus dados
- retificação de dados incorretos
- eliminação de dados
- esclarecimentos adicionais sobre o tratamento de dados

Os pedidos podem ser feitos através do email de suporte indicado nesta política.

8) Eliminação de dados

Os utilizadores podem solicitar a eliminação dos dados associados à utilização da aplicação enviando um pedido para:

app.fs.piloto@gmail.com

O pedido deve indicar que se trata de uma solicitação de eliminação de dados relacionados com a aplicação LASO FS.

Após verificação do pedido, os dados serão eliminados dentro de um prazo razoável, salvo quando exista obrigação legal de conservação.

9) Alterações a esta política

Esta política de privacidade pode ser atualizada periodicamente para refletir alterações na aplicação ou nos requisitos legais.

Sempre que houver alterações relevantes, a data de atualização será revista no topo deste documento.
